# Security

If you discover a vulnerability in this project, please report it responsibly.

- **Preferred:** Open a **private** security advisory on GitHub (repository **Settings → Security → Advisories**), or email the maintainer if contact details are listed on their GitHub profile.
- Include enough detail to reproduce or understand the issue (version, platform, minimal steps).

This project is **Windows-focused** desktop software. Do not commit API keys, tokens, or personal paths to the repository.
